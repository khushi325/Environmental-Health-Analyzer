# 🌍 Personal Environmental Health Analyzer

A responsive web dashboard that analyzes environmental conditions across Indian cities using Air Quality Index (AQI), PM2.5, PM10, and water quality data.

## Features

* Interactive AQI Dashboard
* City Comparison Mode
* Environmental Health Report Card
* Personalized Recommendations
* Data Visualizations using Chart.js
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js


## How to Run

1. Download the project.
2. Open `index.html` in any browser.
